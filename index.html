Hello, I'm Top. Welcome to my website.

  • I'm a 24-year-old coder, movie buff, computer engineer from Bangkok, Thailand, and I'm chiefly interested in technology.

  • A grad student who has an eclectic taste in music is passionate about action figure, RC car, coding, Internet of Things, and photography.



My interest in computer first blossomed when I was in middle school. Because I was really interested in computers, I taught myself how to assemble a personal computer and some basic computer components from scratch. Then, I studied anatomy of a motherboard and installed the components. As a result, I was able to help my friends and cousins building their own computers and chose the computer’s components based on performance requirements and budget. After that, when I was in a high school, I was very interested in a software, so I started creating websites, which I coded in HTML and PHP. Since I enjoyed studying mathematics, physics, and computer, I decided to study Computer Engineering at King Mongkut's Institute of Technology Ladkrabang (KMITL), which is a highly regarded university. At KMITL, I focused largely on computer networks and security. I always enjoyed long hours in the lab because I derived a great deal of benefits regarding practical skills, which I consider to be absolutely essential. However, KMITL did not provide many courses with regard to computer hardware and embedded systems. Consequently, I did not have many opportunities to study computer hardware. However, I remained open to other topics by taking hardware courses such as digital circuit, computer architecture, and computer organization and assembly language. I also used Very High Speed Integrated Circuits (VHSIC) Hardware Description Language (VHDL) to implement my project with a field-programmable gate array (FPGA). In addition, I made an effort to improve my fundamental knowledge and practical skills concerning building a project with Arduino, Raspberri Pi, and FPGA, which I am still mastering. When I was a senior, I had a part-time job as a system operator, and I gained many rewarding experiences about Linux and Unix command, web server configuration, Cisco router and switch configuration, and DNS server configuration. Unfortunately, due to herniated disc, I had back operation and stayed in the hospital and home instead of going to the university. Nevertheless, I wanted graduate at the same time as my friends, so I did not attend to several classes due to a severe back problem. Consequently, I did not get good grade. I successfully carried out my senior project about network security including network security implementation, firewall setup and configuration. I created a website to check IPS and IDS alerts so that users can know the event and immediately what is going on. I also deployed my project to a small company, which is still using this network security system by implementing it with ClearOS. After graduation, I worked as an IT consultant for Deloitte Thailand for nine months. My work mainly concerned cyber security and my responsibility was to find security holes for sensitive data or confidential information for my clients, and report these vulnerabilities to them so that the clients can reduce the threats and protect their private data and intellectual property. My interest in computer hardware pushes me to pursue a Master of Science in Computer Engineering at Syracuse University.
